Kruskal's Algorithm

Description

Kruskal's algorithm finds a maximum or spanning forest of an undirected edge-weighted graph. If the graph is connected, the algorithm finds the maximum or minimum spanning tree.

It uses a greedy algorithm that adds to the forest the lowest/highest-weight that will not form a cycle. Note that the sorting is a part of the algorithm, unless edges are given in a sorted manner.

Time Complexity

\(O(E\log(E))\), which in the case with maximum edges is \(O(E\log(V^2))=O(\) \()\) but most likely the first if you had to pick between them for UIL.

Implementation

Let's look at an implementation for CSES - Road Reparation. Please note that this code will not run in CSES since it is written in Java 25, and CSES uses Java 8. 

import java.util.*;

class DSU{
    int[] parent;
    public DSU(int n) {
        parent = new int[n];
        for(int i = 0; i < n; i++)
            parent[i] = i;
    }
    int find(int a) {
        if(a != parent[a])
            parent[a] = find(parent[a]);
        return parent[a];
    }
    void union(int a, int b) {
        int pa = find(a), pb = find(b);
        if(pa < pb)
            parent[pb] = pa;
        else
            parent[pa] = pb;
    }
}
